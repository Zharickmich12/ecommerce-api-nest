# Backend API - NestJS + MySQL

Este proyecto es una API backend desarrollada con **NestJS**, conectada a una base de datos **MySQL**.  
Incluye autenticación JWT, control de roles, y módulos funcionales para **Usuarios**, **Productos** y **Órdenes**.

---

## Tecnologías principales

- **NestJS** - Framework backend modular y escalable
- **TypeORM** - ORM para interactuar con MySQL
- **MySQL** - Base de datos relacional
- **JWT (JSON Web Token)** - Autenticación basada en tokens
- **Passport** - Middleware de autenticación
- **DBeaver** - Cliente gráfico usado para gestionar la base de datos

## Instalación

Clona el proyecto y entra al directorio:

git clone https://github.com/Zharickmich12/Keycode
cd app-nest

Instala dependencias:
npm install

Ejecuta el proyecto en modo desarrollo:
npm run start:dev

La API estará disponible en:
http://localhost:3000
